About

A sample Angular application built to demonstrate how to read data from a Google Sheet. Since the sheet is just JSON data, this example of course would work using any JSON data source.

This is a PWA, so if you add to home screen (A2HS) when prompted and use that icon, it will open in a standalone window.

Steal any part that may be helpful to you.
